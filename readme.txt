PowerCARD-eCommerce Woocommerce Payment Gateway allows you to accept local and International payment via MasterCard, Visa and Verve Cards.





== Description ==

This is a PowerCARD-eCommerce payment gateway for WooCommerce.

PowerCARD-eCommerce is an payment gateway that allows you to accept payments online.

To get a PowerCARD-eCommerce merchant account visit their website by clicking [here](https://www.PowerCARD-eCommerce.com)

PowerCARD-eCommerce Woocommerce Payment Gateway allows you to accept local and International payment on your Woocommerce store via MasterCard, Visa and Verve Cards.

With this PowerCARD-eCommerce Woocommerce Payment Gateway plugin, you will be able to accept the following payment methods in your shop:

* __MasterCard__
* __Visa__
* __Verve Card__

= Note =

This plugin is meant to be used by merchants.

= Plugin Features =

*   __Accept payment__ via MasterCard, Visa and Verve Cards.
* 	__Seamless integration__ into the WooCommerce checkout page.

== Installation ==

= Automatic Installation =
* 	Login to your WordPress Admin area
* 	Go to "Plugins > Add New" from the left hand menu
* 	In the search box type "PowerCARD-eCommerce Woocommerce Payment Gateway"
*	From the search result you will see "PowerCARD-eCommerce Woocommerce Payment Gateway" click on "Install Now" to install the plugin
*	A popup window will ask you to confirm your wish to install the Plugin.

= Note: =
If this is the first time you've installed a WordPress Plugin, you may need to enter the FTP login credential information. If you've installed a Plugin before, it will still have the login information. This information is available through your web server host.

* Click "Proceed" to continue the installation. The resulting installation screen will list the installation as successful or note any problems during the install.
* If successful, click "Activate Plugin" to activate it.
* 	Open the settings page for WooCommerce and click the "Payment Gateways," tab.
* 	Click on the sub tab for "PowerCARD-eCommerce".
*	Configure your "PowerCARD-eCommerce" settings. See below for details.

= Manual Installation =
1. 	Download the plugin zip file
2. 	Login to your WordPress Admin. Click on "Plugins > Add New" from the left hand menu.
3.  Click on the "Upload" option, then click "Choose File" to select the zip file from your computer. Once selected, press "OK" and press the "Install Now" button.
4.  Activate the plugin.
5. 	Open the settings page for WooCommerce and click the "Payment Gateways," tab.
6. 	Click on the sub tab for "PowerCARD-eCommerce".
7.	Configure your "PowerCARD-eCommerce" settings. See below for details.



= Configure the plugin =
To configure the plugin, go to __WooCommerce > Settings__Â from the left hand menu, then click "Payment Gateways" from the top tab. You should see __"PowerCARD-eCommerce"__ as an option at the top of the screen. Click on it to configure the payment gateway.

__*You can select the radio button next to PowerCARD-eCommerce from the list of payment gateways available to make it the default gateway.*__

* __Enable/Disable__ - check the box to enable PowerCARD-eCommerce Payment Gateway.
* __Title__ - allows you to determine what your customers will see this payment option as on the checkout page.
* __Description__ - controls the message that appears under the payment fields on the checkout page. Here you can list the types of cards you accept.
* __Logo URL__  - enter the full url to your store/site logo. This will be shown on the PowerCARD-eCommerce payment page
* __Public Test Key__  - enter your Public Test Key here. You will get this in your PowerCARD-eCommerce merchant account [PowerCARD-eCommerce](https://www.PowerCARD-eCommerce.com).
* __Private Test Key__  - enter your Private Test Key here. You will get this in your PowerCARD-eCommerce merchant account [PowerCARD-eCommerce](https://www.PowerCARD-eCommerce.com).
* __Public Live Key__  - enter your Public Live Key here. You will get this in your PowerCARD-eCommerce merchant account [PowerCARD-eCommerce](https://www.PowerCARD-eCommerce.com).
* __Private Live Key__  - enter your Private Live Key here. You will get this in your PowerCARD-eCommerce merchant account [PowerCARD-eCommerce](https://www.PowerCARD-eCommerce.com).
* __Test Mode__  - tick to enable test mode.
* Click on __Save Changes__ for the changes you made to be effected.
